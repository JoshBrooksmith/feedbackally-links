# DC7077 FeedbackAlly Survey Links

A mobile-friendly link page for FeedbackAlly surveys, hosted on GitHub Pages.

## 🚀 Live Site

Once deployed: `https://YOUR_USERNAME.github.io/feedbackally-links/`

## ✏️ Editing Survey Links

Edit `index.html` and update the survey links:

```html
<a href="https://feedbackally.walmart.com/survey/YOUR_SURVEY_ID_HERE" ...>
```

Replace `YOUR_SURVEY_ID_HERE` with your actual FeedbackAlly survey IDs.

## 📱 Features

- Mobile-first design
- Walmart blue branding
- WCAG 2.2 AA accessible
- No dependencies, pure HTML/CSS
- Fast loading on any device

## 🛠️ Deployment

1. Push to GitHub
2. Go to repo Settings → Pages
3. Set Source to "Deploy from a branch"
4. Select `main` branch, `/ (root)` folder
5. Save and wait ~1 minute

Your site will be live at `https://YOUR_USERNAME.github.io/feedbackally-links/`
